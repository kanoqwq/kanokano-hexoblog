【教程】frp服务器代理端口转发五分钟快速搭建方法
Created2020-05-05|Updated2020-08-16|教程
|Post View:

所需材料：

  1. 公网linux服务器一台(配置随意，带宽越高越好)
  2. 需要进行端口转发的主机一台

大致思路:服务器端的frp server 提供转发服务，主机客户端使用IP连接到服务器。 宝塔面板下服务端简单配置方法：

  1. 进入宝塔面板的软件商店，搜索frp，下载安装frp服务端
  2. 安装完成后打开，点击frps在线安装
  3. 随后在端口白名单里面添加7000和7001端口，并在宝塔防火墙和服务器提供商的防火墙内放行这两个端口，
  4. 在运行状态一栏启动frp服务即可

frp客户端配置：

  • 下载同版本的客户端：https://github.com/fatedier/frp/releases/download/v0.33.0/frp_0.33.0_windows_amd64.zip
  • 解压
  • 编辑frpc.ini

配置完成后在CMD下执行frpc.exe即可
